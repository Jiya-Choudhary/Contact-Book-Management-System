# 📞 Contact Book Management System

A Python-based contact management application that allows users to search, update, and delete contacts through a simple command-line interface.

## ✨ Features

* 🔍 Search Contacts
* ➕ Add & Update Contacts
* 🗑️ Delete Contacts
* 📖 View Stored Contacts
* ⚡ Menu-Driven Interface

## 🛠️ Tech Stack

* Python 3

## 🚀 How to Run

```bash
python contact_book.py
```

